Magic-symbol-directive

An angularJS directive to consume and display Magic: the Gathering mana costs from the mtgJSON format.

Download this project as a .zip file Download this project as a tar.gz file

An easy way to display mtgJSON!

The project mtgJSON, is a Magic: the Gathering card datasource that is easily read in javascript! Magic-symbol-directive is an AngularJS directive that helps us process this:

	
{ 
	"layout" : "normal", 
	"supertypes" : [  "Legendary" ], 
	"type" : "Legendary Artifact",
	"types" : [  "Artifact" ],
	"colors" : [ ],
	"multiverseid" : 26424, 
	"name" : "Legacy Weapon",
	"cmc" : 7, "rarity" : "Rare",
	"artist" : "John Avon",
 	"manaCost" : "{7}",
 	"text" : "{W}{U}{B}{R}{G}: Exile target permanent
	If Legacy Weapon would be put into a graveyard from anywhere, 
	reveal Legacy Weapon and shuffle it into its owner's library instead.",
 	"number" : "137",
 	"imageName" : "legacy weapon"
}

into this:
